## ADDED Requirements

### Requirement: Goals and Events entries on More
The More tab SHALL provide Goals and Events entries (alongside Recent activity / Upcoming planned style glance rows) that open the in-app Goals list and Upcoming Events list respectively. These are browse hubs, not Settings preference editors.

#### Scenario: Goals from More
- **WHEN** the authenticated user opens More and chooses Goals
- **THEN** the app navigates to the Goals list screen

#### Scenario: Events from More
- **WHEN** the authenticated user opens More and chooses Events
- **THEN** the app navigates to the Upcoming Events list screen
