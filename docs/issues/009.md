# 009 — Zone color ramp (Z1–Z5) across zones UI

**Area:** detail/charts · **Priority:** high · **Effort:** S · **Status:** done

## Problem
Zone tables (planned detail) and zone-time bars (`ActivityCharts`) are monochrome; zones are only distinguishable by reading labels.

## Proposal
Add a zone palette to `src/theme/colors.ts` (conventional blue → green → yellow → orange → red) and document it in `docs/DESIGN.md`. Apply to the zone band list in `app/(app)/planned/[id].tsx`, `BarSeriesChart` zone bars, and the structure silhouette (issue 008).

**OpenSpec change:** `zone-colors-structure-profile`
