# 022 — "Analysis ready" card on Today

**Area:** expansion · **Priority:** medium · **Effort:** S · **Status:** done

## Problem
`WORKOUT_ANALYSIS_READY` only lives as a push; if dismissed, there's no in-app trace that fresh analysis is waiting.

## Proposal
One-line dismissible card on Today ("Yesterday's threshold ride analyzed → view") when the most recent activity has fresh analysis the user hasn't opened. Track "seen" locally; deep-link to the activity screen.
