# 014 — Simplify coach header status line

**Area:** Coach · **Priority:** low · **Effort:** XS · **Status:** open

## Problem
"Live · Tap to switch chats" under the room title is wordy chrome that restates two things visual affordances can carry.

## Proposal
Small colored connection dot next to the room name + a chevron indicating the title is tappable; reserve the status line for real state changes (streaming, polling fallback).
