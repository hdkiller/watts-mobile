# 013 — Render markdown in coach replies

**Area:** Coach · **Priority:** high · **Effort:** M · **Status:** open

## Problem
Assistant bubbles render via plain `<Text>`; AI output containing lists/bold/headers shows literal asterisks and hashes.

## Proposal
Render assistant message text through a lightweight RN markdown renderer (or a minimal in-house parser for bold/lists/inline code) inside `Bubble` in `src/features/coach/CoachChat.tsx`. Style to match chat typography; user bubbles stay plain.
