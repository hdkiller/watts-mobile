# 065 — More → Recent activity / Upcoming: Back lands on Today

**Area:** navigation · **Priority:** high · **Effort:** S–M · **Status:** done

## Problem
More → Training & Schedule pushed into the **Today** tab stack:

- Recent activity → `/(app)/(tabs)/today/activity`
- Upcoming planned → `/(app)/(tabs)/today/upcoming`

Back popped to Today’s root, not More.

## Proposal
Hoist multi-entry list/detail destinations to the root app stack so Back returns to the opener.

## Resolution (2026-07-23)
Moved activity, planned, upcoming, and events onto the root `(app)` stack (`APP_HREFS.activityList` → `/(app)/activity`, etc.). `migrateLegacyAppHref` rewrites old Today-tab hrefs from push/pending returns.
