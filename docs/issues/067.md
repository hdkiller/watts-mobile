# 067 — Athlete → Sports settings: Back abandons the root Athlete screen

**Area:** navigation / profile · **Priority:** medium · **Effort:** S · **Status:** open

## Problem
Root-stack Athlete (`/(app)/athlete`) links “Sports” to `/(app)/(tabs)/more/settings/sports`. Opening that jumps into the More Settings stack; Back returns to Settings → More, not Athlete. Undoes the root-stack Back win for the inline sports affordance.

## Proposal
Root-stack sports list (or nest under athlete), or a custom Back that returns to `APP_HREFS.athlete`. Prefer a shared destination if Settings → Sports must stay the IA home.
