# 012 — Coach input bar: icons + native attach sheet

**Area:** Coach · **Priority:** medium · **Effort:** S · **Status:** done

## Problem
The "Photo" and "Send" text buttons make the input bar wide and unlike every messenger; attach uses `Alert.alert`, which looks like an error dialog on Android.

## Proposal
In `src/features/coach/CoachChat.tsx`: replace "Photo" with a camera/plus glyph and "Send" with a send-arrow icon (SF Symbols / emoji fallback, à la `SportIcon`), keep accessibility labels. Use `ActionSheetIOS` on iOS (or a small bottom sheet) for the camera/library choice.

## Resolution
Shipped in `feat/coach-chat-ux-polish`: circular plus / arrow-up icon buttons; iOS `ActionSheetIOS` and Android bottom-sheet modal for attach. Device visual check optional.
