# 008 — Workout structure silhouette on planned detail

**Area:** detail · **Priority:** high · **Effort:** M · **Status:** open

## Problem
`structureSteps` renders as a text list only. The compact intensity-profile graphic (colored blocks, width = duration, height/color = intensity) is the visual signature athletes expect and instantly communicates the session shape.

## Proposal
New `StructureProfile` component (SVG via `react-native-svg`, already installed) rendered above the step list in `app/(app)/planned/[id].tsx`. Reuse the zone color ramp from issue 009. Data: `durationSec` + `intensityLabel`/zone per step from `mapActivity.ts`.

**OpenSpec change:** `zone-colors-structure-profile`
