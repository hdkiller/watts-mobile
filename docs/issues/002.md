# 002 — Haptic feedback on key moments

**Area:** cross-cutting · **Priority:** medium · **Effort:** S · **Status:** open

## Problem
Accepting a recommendation, saving a check-in, and analysis completion give no tactile response; the app feels flat for a field companion.

## Proposal
Add `expo-haptics`. Light impact on primary CTA press, `notificationAsync(Success)` on accept/save success, error haptic on failed mutation. Centralize in the shared `Button` (optional `haptic` prop) plus explicit calls at mutation success points.
