# 020 — Plan-vs-done compliance marks on lists

**Area:** expansion · **Priority:** medium · **Effort:** M · **Status:** open

## Problem
Recently/Coming up rows don't show whether a planned session was actually completed as planned — the lists feel raw rather than coached.

## Proposal
Pair completed workouts with planned ones (same day + type, or backend linkage if available) and render a small mark: green check (done as planned), amber tilde (modified), grey dash (missed). Start in `recently-teaser.tsx` and `activity/index.tsx`.
