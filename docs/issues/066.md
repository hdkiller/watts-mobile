# 066 — Today / activation → Health Sync & Connected Apps: Back leaves the setup path

**Area:** navigation / activation · **Priority:** high · **Effort:** M · **Status:** done

## Problem
Finish setup, Connect a device, and activation connect pushed into More → Settings. Back walked Settings → More instead of Today / the wizard.

## Proposal
Present Health Sync / Connected Apps as root-stack screens when opened outside More.

## Resolution (2026-07-23)
Moved Health Sync (+ history / recent workouts) and Connected Apps to the root stack:

- `/(app)/health-sync`, `/(app)/health-history`, `/(app)/health-workouts`
- `/(app)/connected-apps`

`APP_HREFS.settingsHealth` / `settingsConnectedApps` point there. Settings hub and Finish setup / activation share the same destinations; Back returns to the opener.
