# 018 — Login polish: hide dev plumbing, add wordmark

**Area:** auth · **Priority:** high · **Effort:** S · **Status:** open

## Problem
`app/(auth)/login.tsx` shows the OAuth redirect URI and a `pnpm cw:cli` command to every user — reads as broken/scary in a store build. The first screen has no brand presence.

## Proposal
Gate the redirect-URI/CLI block behind `__DEV__` or Expo Go detection (or an "Advanced" disclosure). Add the Coach Watts logo/wordmark above the title.

**OpenSpec change:** `login-and-error-polish`
