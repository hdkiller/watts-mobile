# 021 — Day-grouped Upcoming list

**Area:** expansion · **Priority:** low · **Effort:** S · **Status:** open

## Problem
Upcoming is a flat list; scanning "what's tomorrow vs Thursday" requires reading every date.

## Proposal
Group `app/(app)/upcoming/index.tsx` rows under "Today / Tomorrow / Thu Jul 23" section headers (FlatList sections or SectionList); optionally show rest days as quiet rows. A lightweight week view without building a calendar (heatmaps stay a non-goal).
