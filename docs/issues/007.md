# 007 — Log screen cleanup: remove "Back to Today", icon for read-only chips

**Area:** Log · **Priority:** low · **Effort:** XS · **Status:** done

## Problem
A full-width "Back to Today" button duplicates the tab bar. Read-only recovery chips append the word "· view", which is cryptic inside a chip.

## Proposal
Remove the button at the bottom of `app/(app)/(tabs)/log.tsx`. Replace the "· view" suffix in `RecoveryChip` with a small lock/eye glyph (or drop it entirely).

## Done
- Removed "Back to Today" button from Log
- Read-only `RecoveryChip` shows a small eye glyph (SF Symbol on iOS, emoji fallback elsewhere) instead of "· view"
