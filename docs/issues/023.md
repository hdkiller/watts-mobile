# 023 — Home-screen widget / Live Activity for today's session

**Area:** expansion · **Priority:** high · **Effort:** L · **Status:** done

## Problem
"What am I doing today?" currently requires opening the app — the highest-leverage companion surface is the home screen.

## Proposal
iOS widget (and Android equivalent) showing today's session name + duration + rest/train state, refreshed from the Today payload. Requires a native widget target (expo-apple-targets or config plugin) and a shared data container; plan as its own change with design + native-module doc updates.
