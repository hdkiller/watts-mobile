# 017 — Unread badge on More tab icon

**Area:** More · **Priority:** low · **Effort:** XS · **Status:** done

## Problem
`useUnreadNotificationsCount` is only visible after opening More; unread notifications are undiscoverable from the tab bar.

## Proposal
Surface the count as a badge on the More trigger in `app/(app)/(tabs)/_layout.tsx` (NativeTabs supports badges). Cap display at "9+".

## Resolution
Shipped in `feat/more-tab-polish`: `NativeTabs.Trigger.Badge` on the More tab from `useUnreadNotificationsCount`, displaying the count or `9+` when above 9.
