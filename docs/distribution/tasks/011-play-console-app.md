**Area:** listing · **Priority:** medium · **Status:** open

**Depends on:** [010](./010-google-play-developer-account.md)

## Goal

Create the Play Console application so AAB uploads and listing drafts can proceed.

## Steps

1. [ ] Play Console → Create app → name **Coach Watts**, default language English.
2. [ ] App or game: **App**. Free/paid: Free (unless product decides otherwise).
3. [ ] Declarations: privacy policy URL `https://coachwatts.com/privacy`; confirm not a government / news special category unless true.
4. [ ] Package name must be **`com.coachwatts.mobile`** (matches `app.json` `android.package`) — set on first upload / app creation; cannot change later.
5. [ ] Note Play app id / package in [log.md](../log.md).

## Done when

- App exists in Play Console; package reserved as `com.coachwatts.mobile`.
